
# 仿抖音视频流

## 说明
- 版本：1.0.0

## 开发目录
```js
├─common           // 公共函数方法
├─pages            // vue所有页面目录
│  ├─douyin         // 
│  │   └─index.vue
├─static          // 静态资源
├─template.h5.html// 自定义html模板
├─main.js         // js入口文件
├─App.vue         // 页面入口文件
├─manifest.json   // 应用的配置文件
├─pages.json      // 全局配置，决定页面文件的路径、窗口样式、原生的导航栏、底部的原生tabbar 等
├─package.json    // 配置
└─uni.scss        // css
```


## 打包目录（不可自定义）
```js
\douyin\unpackage\dist\build\h5
```
